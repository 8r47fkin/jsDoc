Literals


A literal is a data value that appears directly in a program. The following are all literals: 12 // The number twelve 1.2 // The number one point two "hello world" // A string of text 'Hi' // Another string true // A Boolean value false // The other Boolean value null // Absence of an object Complete details on numeric and string literals appear in Chapter 3.
